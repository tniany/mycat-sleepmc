### 项目简介
睡覺吧mc伺服器 — 一款使Minecraft伺服器“休眠”的小程序，隶属于浅月不息-mc工具箱系列，旨在通过多种技术手段对Minecraft服务器进行相关操作，适用于特定场景下的服务器管理与测试需求。

### 版本信息

当前最新版本：v2.03.6

### 版本更新日志

#### v2.03.6 新增特性

- 支持顶号强改uuid：通过强改uuid的方式，使该玩家无法通过原有账号登录至目标服务器，实现针对性的登录限制效果。

- 支持伪MODbot登录：针对部分需下载对应MOD才能进入的服务器，新增伪MOD模拟功能，可使服务器误认为机器人已安装所需MOD，从而成功绕过MOD验证机制。

#### 旧版本核心功能

- 假人bot攻击：模拟大量虚拟玩家（bot）对服务器发起连接请求，占用服务器资源。

- motd攻击：针对服务器的MOTD（服务器状态信息）展示功能发起攻击，干扰服务器正常的状态反馈。

- 高版本漏洞攻击：利用Minecraft高版本服务器的已知漏洞，对服务器进行针对性操作。

- dd攻击：通过分布式拒绝服务攻击手段，影响服务器的正常网络连接与服务响应。

- 持续顶号：持续对目标玩家账号发起登录请求，干扰该玩家正常登录服务器。

- 伪装op：模拟服务器管理员（op）身份，尝试获取相关操作权限或干扰服务器管理。

- 假数据包返回：通过发送伪造的数据包，有效绕过服务器的bot检测和无mod检测机制，提升操作的隐蔽性与成功率。
